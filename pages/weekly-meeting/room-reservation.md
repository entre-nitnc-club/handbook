# 講義室予約ガイド

アントレプレナークラブでは、毎週月曜日の16:00〜18:00に定例会を開催しています。
定例会はインキュベーションルームで行われるため、事前にインキュベーションルームの予約が必要です。

## 定例会のインキュベーションルーム予約方法

[フォーム(校舎等使用願・課外教育活動施設使用許可願)](https://forms.office.com/r/kEMcLU8THV)を開く

| 欄名 | 回答内容 |
|---|---|
| 使用責任者氏名 | (自分の名前) |
| 学科・専攻 | (自分の学科) |
| 学年 | (自分の学年) |
| 使用目的 | 部活動 |
| 団体種別 | 文化部 |
| 団体名を選択（文化部） | アントレプレナークラブ |
| 講義室を使用しますか？ | いいえ |

インキュベーションルームは、*講義室ではない*ので注意してください。

| 欄名 | 回答内容 |
|---|---|
| 使用施設種別 | 課外活動教育施設 |
| 使用施設（課外活動教育施設） | その他, インキュベーションルーム |
| 使用日時 | (活動日) |
| 使用開始時間 | 16:00 |
| 使用終了時間 | 18:00 |
| 使用人数 | 30(部員数に合わせて適宜調整) |
| 部室の使用有無 | 無 |

インキュベーションルームは、*課外活動教育施設*となります。
また使用施設はその他を選択し、インキュベーションルームと入力してください。

使用人数はその日の使用人数に合わせてください。(基本的には部員数を大体で記入)

備考欄には、準備時間などを設け*早く部屋を借りる場合*や、講義室など学生課で*仮予約が必要な部屋を借りる場合*は、その旨を記入してください。

以上で、インキュベーションルームの予約は完了です。
予約が完了したら、定例会の準備を進めましょう。
